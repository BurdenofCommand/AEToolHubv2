# Astro Empires — BBCode Reference (v5.0)

*Compiled from: BBCode.pdf*
*(Source page retrieved 25 Jun 2026; navigation chrome stripped for readability)*

BBCode is a special implementation of HTML, similar in style but with tags enclosed in square brackets `[ ]` rather than `< >`. Use it when sending in-game messages or posting on your guild board.

## Basic Style Tags

| Style | Tag | Example |
|---|---|---|
| Bold | `[b]Bold[/b]` | **Bold** |
| Underline | `[u]Underline[/u]` | Underline |
| Italic | `[i]Italic[/i]` | *Italic* |
| Strikethrough | `[s]Strikethrough[/s]` | ~~Strikethrough~~ |

## Text Color

Use a recognized color name (e.g. `red`) or a hex triplet (e.g. `#FFFFFF`):

```
[color=red]Astro Empires[/color]
[color=#FF0000]Astro Empires[/color]
```

## Text Size

Range: 1 (small) to 9 (large).

```
[size=1]Astro Empires[/size]
[size=5]Astro Empires[/size]
```

## Fixed Width Text

```
[code]Astro Empires[/code]
```

## Lists

```
[list]
[*]Fighters
[*]Bombers
[*]Cruiser
[/list]
```
Renders as:
- Fighters
- Bombers
- Cruiser

## Links

```
[url=https://forum.astroempires.com]Visit Astro Empires Forum![/url]
[url]https://forum.astroempires.com[/url]
```

## Images

```
[img]https://www.astroempires.com/images/ae_logo.png[/img]
```
Note: `[img]` cannot be used in private messages during the first day of a game account.

## Text Alignment

```
[center]Astro Empires[/center]
[left]Astro Empires[/left]
[right]Astro Empires[/right]
```

## Restrictions

Images and Text Size BBCode tags are disabled in Player/Guild Profile descriptions.
